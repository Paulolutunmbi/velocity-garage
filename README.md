# Velocity Garage

Velocity Garage is a production-ready multi-page vanilla JavaScript app for browsing and managing a curated supercar catalog. It uses Firebase Authentication, Firestore, and Storage for identity, user state, and profile media.

## Core Features

- Public landing experience with gated access to authenticated pages
- Email/password authentication, Google sign-in, and GitHub sign-in
- Auth-guarded user pages:
  - Home showroom
  - Favorites
  - Wishlist
  - Compare (max 3 cars)
  - Profile management
- Runtime recommendation panel on the home page
- Admin dashboard with live analytics and user management
- Theme preference persistence (dark/light) per user

## Environment

Runtime client config is loaded from env.js (window.__ENV), generated by the build step.

Required values:

- FIREBASE_API_KEY
- FIREBASE_AUTH_DOMAIN
- FIREBASE_PROJECT_ID
- FIREBASE_STORAGE_BUCKET
- FIREBASE_MESSAGING_SENDER_ID
- FIREBASE_APP_ID
- FIREBASE_ADMIN_EMAIL

## Local Development

1. Install dependencies

npm install

2. Create local env file (example)

Copy-Item .env.example .env.local

3. Generate runtime env

npm run build

4. Start local server

npm run dev

Default local URL: http://127.0.0.1:5000

## Scripts

- npm run build
  - Generates env.js from environment variables
- npm run dev
  - Runs build, then serves static files on port 5000

## Deployment

- vercel.json is configured for static deployment
- buildCommand: npm run build
- outputDirectory: .
